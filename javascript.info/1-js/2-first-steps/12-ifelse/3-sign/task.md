# Получить знак числа

[importance 2]

Используя конструкцию `if..else`, напишите код, который получает значение `prompt`, а затем выводит `alert`:
<ul>
<li>`1`, если значение больше нуля,</li>
<li>`-1`, если значение меньше нуля,</li>
<li>`0`, если значение равно нулю.</li>
</ul>
