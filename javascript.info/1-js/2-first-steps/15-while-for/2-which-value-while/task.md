# Какие значения i выведет цикл while?

[importance 4]

Для каждого цикла запишите, какие значения он выведет. Потом сравните с ответом.
<ol>
<li>Префиксный вариант

```js
var i = 0;
while (++i < 5) alert( i );
```

</li>
<li>Постфиксный вариант

```js
var i = 0;
while (i++ < 5) alert( i );
```

</li>
</ol>
