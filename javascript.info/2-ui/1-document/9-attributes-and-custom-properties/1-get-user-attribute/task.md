# Получите пользовательский атрибут

[importance 5]

<ol>
<li>Получите `div` в переменную.</li>
<li>Получите значение атрибута `"data-widget-name"` в переменную.</li>
<li>Выведите его.</li>
</ol>

Документ:

```html
<body>

  <div id="widget" data-widget-name="menu">Выберите жанр</div>

  <script>
    /* ... */
  </script>
</body>
```
