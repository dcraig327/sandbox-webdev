# Создайте дерево из объекта

[importance 5]

Напишите функцию, которая создаёт вложенный список `UL/LI` (дерево) из объекта.

Например:

```js
var data = {
  "Рыбы":{
    "Форель":{},
    "Щука":{}
  },

  "Деревья":{
    "Хвойные":{
      "Лиственница":{},
      "Ель":{}
    },
    "Цветковые":{
      "Берёза":{},
      "Тополь":{}
    }
  }
};
```

Синтаксис:

```js
var container = document.getElementById('container');
*!*
createTree(container, data); // создаёт 
*/!*
```
