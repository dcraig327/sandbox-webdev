# Шаблон для таблицы с пользователями

[importance 5]

```js
var users = [
  {name: "Вася", age: 10},
  {name: "Петя", age: 15},
  {name: "Женя", age: 20},
  {name: "Маша", age: 25},
  {name: "Даша", age: 30},
];
```

Выведите их в виде таблицы `TABLE/TR/TD` при помощи шаблона.
